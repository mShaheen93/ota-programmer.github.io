OTA PROGRAMMER

For STM32 Microcontroller

On The Air Programmer developed to write .hex file in the Flash Memory of STM32 MCU. The Programmer uses NodeMCU Board to recieve the .hex file from the website and sends it to STM via UART to write the code using the Boot Loader flashed on STM.
